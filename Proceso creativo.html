Proceso creativo

Aqui podrás ver la evolución de nuestro proyecto

LLuvia de ideas

Mediante esta lluvia de ideas proponemos palabras que sean esencia de nuestro proyecto o de aquello que nosotros buscamos realizar.

Luego de llegar a una cantidad de palabras buscamos hacer una hilación y de ver cuales unidas pueden ayudarnos a formar un concepto.

Bocetos

En este proceso se busca una idea gráfica que nos ayude a visualizar nuestro proyecto para así poder darle una proporción al prototipo.

En este boceto se pueden adjuntar anotaciones que indique las funciones que tendra el prototipo.

Simulación

En esta parte del proceso se ve el diseño de circuito y se hacen pruebas para no comprar componentes en vano o en el caso contradictorio comprarlos y no usarlos correctamente por un error en el cálculo de voltaje o no poniendo un componente que tenga el valor adecuado para que todo funcione de manera correcta

Thinkercad

Este programa fue el que utilizamos para poder hacer simulaciones de nuestro proyecto y entender el funcionamiento de algunos componentes que el programa nos facilita para poder usarlos de manera de prueba.

Simulación 1

Para poder hacer una simulación básica usamos uno de los ejemplos que viene dentro del programa del arduino IDE

Armado de circuito

Luego de haber entendido lo que programamos en arduino ide procedemos a hacer el armado del circuito con componentes que nos brindo el profesor

Programación

Después de colocar los componentes de manera correcta, revisar tambien las conecciones positivas y negativas y ver los puertos que se estan usando en el arduino se procede a programar poniendo los valores de los puertos usados y luego se carga lo programado al arduino.

Falla de circuito

Al cargar el programa al arduino notamos que el foco led no encendia asi que tuvimos que pedir asesoria al profesor para poder entender el porque no encendia el foco led, el problema fue que usamos una resistencia que no tenía el valor requerido que necesitaba el foco led, menos mal el foco no se quemó. Asi que el profesor nos ayudo a hacer de nuevo el circuito.

Prueba exitosa

Luego de que el profesor nos ayudo a poder corregir el circuito se volvió a cargar la programación en el arduino y así pudimos encender de manera correcta un foco led.
